# 📦 Sistema de Gestión Logística de Inventarios para Microempresas

Sistema web desarrollado para optimizar la gestión logística de inventarios en microempresas, permitiendo controlar productos, ventas y movimientos de stock de manera eficiente y organizada.

---

## 📖 Descripción

La aplicación facilita la administración integral del inventario mediante herramientas que permiten registrar productos, controlar existencias, gestionar ventas y generar reportes para la toma de decisiones.

El sistema está diseñado para pequeñas empresas que requieren un control preciso del stock disponible y un seguimiento detallado de las operaciones comerciales.

---

## 🎯 Objetivo

Optimizar el control de inventarios y ventas en microempresas mediante una plataforma web que permita:

- Administrar productos y categorías.
- Monitorear existencias en tiempo real.
- Registrar movimientos de inventario.
- Gestionar clientes y proveedores.
- Controlar ventas y devoluciones.
- Generar reportes e indicadores para apoyar la toma de decisiones.

---

# 👥 Roles del Sistema

## 🔐 Administrador

Cuenta con acceso total a las funcionalidades del sistema.

### Funcionalidades

- 👤 Gestión de usuarios.
- 📦 Gestión de productos.
- 🗂️ Gestión de categorías.
- 🚚 Gestión de proveedores.
- 🤝 Gestión de clientes.
- 📋 Gestión de inventario.
- ⚠️ Control de stock mínimo y máximo.
- 💲 Actualización de precios.
- 🛒 Consulta de ventas.
- 🔄 Registro de ajustes de inventario.
- ↩️ Gestión de devoluciones.
- 📊 Generación de reportes.
- 📈 Dashboard con indicadores de gestión.

---

## 🛍️ Vendedor

Encargado del proceso de ventas y atención al cliente.

### Funcionalidades

- 🛒 Registrar ventas.
- 🔎 Buscar productos.
- 📦 Consultar stock disponible.
- 🤝 Consultar información de clientes.
- 🔢 Registrar cantidades vendidas.
- 🧾 Consultar historial de ventas propias.

---

# 🧩 Módulos del Sistema

| Módulo | Descripción |

👤 Usuarios: Autenticación y administración de  usuarios.

📦 Productos: Registro, actualización y consulta de productos. 

🗂️ Categorías: Clasificación de productos por grupos. 

📋 Inventario: Control de entradas, salidas y existencias.

🛒 Ventas: Registro de ventas realizadas. 

🤝 Clientes: Administración de información de clientes. 

🚚 Proveedores: Gestión de proveedores del negocio. 

📊 Reportes  Generación de informes de inventario y ventas.
