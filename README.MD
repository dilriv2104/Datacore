Sistema de Gestión Logística de Inventarios para Microempresas

DESCRIPCION

Sistema web desarrollado para facilitar la gestión logística de inventarios en microempresas. Permite controlar productos, categorías, proveedores, clientes, ventas y movimientos de inventario, garantizando un seguimiento eficiente del stock disponible.

El sistema cuenta con dos roles principales:

-Administrador: acceso completo a todas las funcionalidades del sistema.

-Vendedor: encargado de registrar ventas y consultar información necesaria para la atención de clientes.

 OBJETIVO

Optimizar el control de inventarios y ventas en microempresas mediante una plataforma web que permita administrar productos, monitorear existencias y generar registros de movimientos de manera organizada.

FUNCIONALIDADES

ADMINISTRADOR:

-Gestión de usuarios.
-Gestión de productos.
-Gestión de categorías.
-Gestión de proveedores.
-Gestión de clientes.
-Gestión de inventario.
-Control de stock mínimo y máximo.
-Actualización de precios.
-Consulta de ventas.
-Registro de ajustes de inventario.
-Gestión de devoluciones.
-Generación de reportes.
-Visualización de dashboard con indicadores.

VENDEDOR

-Registrar ventas.
-Buscar productos.
-Consultar stock disponible.
-Consultar clientes.
-Registrar cantidades vendidas.
-Consultar historial de ventas propias.

MODULOS DEL SISTEMA

USUARIOS
-Permite la autenticación y administración de usuarios del sistema.

PRODUCTOS
-Permite registrar, actualizar y consultar productos.

CATEGORIAS
-Permite clasificar los productos por grupos.

INVENTARIO
-Controla las entradas, salidas y existencias de productos.

VENTAS
-Registra las ventas realizadas por los vendedores.

CLIENTES
-Administra la información de los clientes.

PROVEEDORES
-Gestiona los proveedores que suministran los productos.

DEVOLUCIONES
-Permite registrar devoluciones de productos vendidos.

REPORTES
-Genera informes sobre inventario, ventas y movimientos.
